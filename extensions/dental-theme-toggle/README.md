# Dental Theme Toggle Extension

A specialized OHIF extension that provides theme toggle functionality for dental imaging workflows.

## Features

- Theme switching between OHIF and Dental modes
- Dental-optimized color schemes and UI
- Persistent theme preference
- Panel component integration
- Available commands for programmatic control

## Usage

This extension is automatically integrated with the dental mode and provides a theme toggle panel in the viewer interface.

## Available Commands

- `setViewerTheme` - Set theme to 'ohif' or 'dental'
- `toggleViewerTheme` - Toggle between themes
- `getViewerTheme` - Get current theme setting

## CSS Classes Applied

- `.ohif-theme` - Standard OHIF viewer styling
- `.dental-theme` - Dental-optimized viewer styling
