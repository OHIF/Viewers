# Icon

## Basic usage

<Playground>
  <div className="flex flex-wrap items-center content-center color-white">
    {Object.keys(ICONS)
      .sort()
      .map((name, i) => {
        return (
          <React.Fragment key={i}>
            <div className="flex flex-col m-1 p-4 items-center rounded">
              <Icon name={name} />
              <div className="break-normal">{name}</div>
            </div>
          </React.Fragment>
        );
      })}
  </div>
</Playground>

## API

<Props of={Icon} />

The `Icon` component takes a single prop of `name`, which indicates which Icon
should be displayed. Again, the icon responds to contextual text/font styles.
